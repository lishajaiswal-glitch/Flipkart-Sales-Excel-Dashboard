# Flipkart Sales Excel Dashboard

## Project Overview
This project is an interactive Excel dashboard created to analyze Flipkart sales data and customer purchasing trends.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers
- Conditional Formatting

## Key Insights
- Sales performance analysis
- Category-wise revenue trends
- Customer purchasing behavior
- Profit and order analysis
